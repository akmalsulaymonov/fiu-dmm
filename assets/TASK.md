# Редизайн сайта Департамента финансового мониторинга при НБТ
### Аналитический бриф + фирменный стиль + промпт для Claude Design

---

## 1. Что такое DMM и анализ текущего сайта

**Департамент финансового мониторинга при Национальном банке Таджикистана (DMM)** — это подразделение финансовой разведки страны (Financial Intelligence Unit, FIU; в русскоязычной традиции — ПФР). Его задачи: сбор и анализ информации о подозрительных финансовых операциях, противодействие легализации (отмыванию) доходов, финансированию терроризма и финансированию распространения оружия массового поражения (ПОД/ФТ/ФРОМП), взаимодействие с поднадзорными организациями (банки, НКО, страховщики), методическое обеспечение и обучение, а также международное сотрудничество (ФАТФ, ЕАГ, Эгмонтская группа).

### Текущее состояние (`nbt.tj/dmm/ru/`)

Сайт является поддоменом/подразделом портала НБТ и наследует его устаревший шаблон (Bitrix-вёрстка `nbt_new_temp`). Ключевые проблемы:

| Проблема | Описание |
|---|---|
| **Визуальная устарелость** | Плотные текстовые блоки, мелкая типографика, нет воздуха, табличная вёрстка |
| **Слабая навигация** | Глубокая вложенность, неочевидная структура, дублирование пунктов меню |
| **Отсутствие иерархии задач** | Поднадзорные организации, граждане и партнёры не разведены по сценариям |
| **Нет фокуса на действиях** | Подача отчётов, реестры, формы обратной связи спрятаны в тексте |
| **Контент-перегруз главной** | Курсы валют, ставки, новости банка — нерелевантны профилю ПФР |
| **Доступность** | Слабый контраст, нет полноценной версии для слабовидящих, нет адаптива |
| **Многоязычность** | Тоҷикӣ / Русский / English реализованы, но непоследовательно |

### Лучшие практики ПФР других стран (ориентиры)

- **Росфинмониторинг (fedsfm.ru)** — чёткое разделение «Гражданам / Организациям / Сотрудничество», публичный перечень террористов и экстремистов.
- **АФМ Казахстана** — современный портал с разделами для субъектов финмониторинга и онлайн-сервисами.
- **AUSTRAC (Австралия)** — образцовая структура «для бизнеса / для отчётности / руководства» и сильный гайд по комплаенсу.
- **FinCEN (США)** — раздел регуляторики, BSA E-Filing, advisories/alerts.
- **ДФМ Беларуси, FIU-нетворк Эгмонтской группы** — единый стандарт раскрытия задач, функций, правовой базы.

Общий вывод: современный ПФР-сайт строится вокруг **задач пользователя** (подать отчёт, проверить реестр, изучить требования), а не вокруг оргструктуры.

## 2. Структура и разделы нового сайта (информационная архитектура)

Навигация строится по трём аудиториям: **поднадзорные организации**, **граждане/бизнес**, **партнёры/международное сотрудничество**.

```
ГЛАВНАЯ
│
├── О Департаменте
│   ├── О нас (миссия, мандат)
│   ├── Задачи и функции
│   ├── Правовой статус и полномочия
│   ├── Структура и руководство
│   ├── История создания
│   └── Контакты
│
├── Субъектам финмониторинга (поднадзорным организациям)   ← ключевой раздел
│   ├── Кто является субъектом ПОД/ФТ/ФРОМП
│   ├── Постановка на учёт / регистрация
│   ├── Порядок направления сообщений (отчётности)
│   ├── Личный кабинет / электронная подача ▸ (сервис)
│   ├── Требования к внутреннему контролю и комплаенсу
│   ├── Методические рекомендации и типологии
│   ├── Обучение (Учебно-методический центр)
│   └── Часто задаваемые вопросы
│
├── Правовая база
│   ├── Законы РТ (ПОД/ФТ/ФРОМП)
│   ├── Нормативные акты НБТ и DMM
│   ├── Рекомендации ФАТФ (40 рекомендаций)
│   ├── Международные конвенции и стандарты
│   └── Проекты нормативных актов (общественное обсуждение)
│
├── Противодействие (направления деятельности)
│   ├── Отмывание доходов (ПОД)
│   ├── Финансирование терроризма (ФТ)
│   ├── Финансирование ОМП (ФРОМП)
│   ├── Перечень террористов и экстремистов / санкционные списки
│   ├── Национальная оценка рисков (NRA)
│   └── Финансовые расследования и типологии
│
├── Международное сотрудничество
│   ├── ФАТФ и региональные органы (ЕАГ / APG)
│   ├── Эгмонтская группа
│   ├── Двусторонние соглашения
│   └── Взаимная оценка (Mutual Evaluation) и отчёты
│
├── Пресс-центр
│   ├── Новости и события
│   ├── Пресс-релизы
│   ├── Публикации, отчёты, бюллетени
│   ├── Видео и медиа
│   └── Анонсы обучающих мероприятий
```

### Структура главной страницы (секции сверху вниз)

1. **Хедер** — герб/лого DMM + НБТ, переключатель языков (ТҶ/РУ/EN), кнопка «Версия для слабовидящих», глобальный поиск, CTA «Личный кабинет субъекта».
2. **Hero** — короткий слоган о миссии + 2 главных CTA: «Подать сообщение» и «Изучить требования». Спокойный фон (паттерн/абстракция, не фото-сток).
3. **Быстрые сценарии (3 карточки)** — «Я субъект финмониторинга» / «Я гражданин» / «Я партнёр». Каждая ведёт в свою воронку.
4. **Направления деятельности** — ПОД / ФТ / ФРОМП в виде иконографических карточек.
5. **Перечни и реестры** — акцентный блок с прямым доступом к санкционным спискам и реестру субъектов.
6. **Новости и события** — 3–4 последних, минималистичные карточки.
7. **Правовая база / документы** — подборка ключевых законов и рекомендаций ФАТФ.
8. **Международное сотрудничество** — логотипы/ссылки ФАТФ, ЕАГ, Эгмонт.
9. **Футер** — контакты, телефон доверия, эл. обращение, соцсети, ссылка на головной сайт НБТ, правовая информация.

## 3. Фирменный стиль (Design System / дизайн-токены)

Стиль должен передавать **доверие, государственность, безопасность и современность**. Опора на палитру НБТ (глубокий синий) с собственным акцентом DMM.

### 3.1. Цветовая палитра

```css
:root {
  /* — Базовые (доверие, государственность) — */
  --dmm-navy-900:   #0A2540;  /* основной тёмно-синий, хедер/футер */
  --dmm-navy-800:   #103A66;  /* фон секций, акценты */
  --dmm-blue-600:   #1E5BA8;  /* основной бренд-синий (кнопки, ссылки) */
  --dmm-blue-500:   #2D6FCB;  /* hover/active */
  --dmm-blue-100:   #E6EEF7;  /* светлый фон карточек */

  /* — Акцент DMM (безопасность/внимание) — */
  --dmm-teal-600:   #0E8A7D;  /* фирменный акцент, иконки, индикаторы */
  --dmm-teal-400:   #19B4A3;

  /* — Функциональные (статусы, оповещения) — */
  --dmm-alert-600:  #C2410C;  /* предупреждения, санкционные списки */
  --dmm-success:    #15803D;  /* успешные статусы */
  --dmm-gold-500:   #C9A227;  /* геральдический акцент (герб/награды) */

  /* — Нейтральные — */
  --dmm-ink-900:    #14202E;  /* основной текст */
  --dmm-ink-600:    #475569;  /* вторичный текст */
  --dmm-line-200:   #E2E8F0;  /* границы, разделители */
  --dmm-bg-50:      #F7F9FC;  /* фон страницы */
  --dmm-white:      #FFFFFF;
}
```

**Принципы применения**
- Тёмно-синий `navy-900` — хедер, футер, «государственные» секции.
- `blue-600` — все интерактивные элементы (кнопки, ссылки).
- `teal-600` — фирменный акцент DMM, отстраивает департамент от головного НБТ.
- `alert-600` — только для перечней террористов/экстремистов и важных предупреждений.
- Соотношение цветов ≈ 70% нейтральные / 20% синий / 10% акценты.

### 3.2. Типографика

```css
/* Заголовки — строгий, «институциональный» гротеск с поддержкой кириллицы и таджикской графики */
--font-display: "PT Root UI", "Manrope", "Inter", system-ui, sans-serif;

/* Основной текст — высокая читаемость, полная кириллица */
--font-body: "Inter", "PT Sans", "Roboto", system-ui, sans-serif;

/* Шкала (desktop) */
--fs-hero:   clamp(2.2rem, 4vw, 3.5rem);  /* 700, line-height 1.1 */
--fs-h1:     2.25rem;   /* 700 */
--fs-h2:     1.75rem;   /* 600 */
--fs-h3:     1.25rem;   /* 600 */
--fs-body:   1.0625rem; /* 400, line-height 1.6 */
--fs-small:  0.875rem;
```

- **Поддержка трёх языков обязательна:** все шрифты должны корректно отображать таджикскую кириллицу (ҷ, ӣ, ӯ, ҳ, ғ, қ). PT Root UI / PT Sans (Paratype) гарантируют это.
- Заголовки — плотные, без курсива; текст — свободный line-height (1.6) для длинных правовых текстов.

### 3.3. Компоненты и стилистика

- **Скругления:** умеренные — `border-radius: 10px` для карточек, `8px` для кнопок (солидно, не «игрушечно»).
- **Тени:** мягкие, низкие — `box-shadow: 0 1px 3px rgba(10,37,64,.08)`.
- **Кнопки:** заливка `blue-600`, текст белый; вторичные — обводка `blue-600`. Чёткие состояния hover/focus (важно для доступности).
- **Карточки:** белый фон, тонкая граница `line-200`, иконка-акцент сверху.
- **Иконки:** линейные, единый набор (Lucide / Phosphor), 1.5px обводка.
- **Сетка:** 12 колонок, контейнер max-width 1200px, отступы кратны 8px.
- **Геральдика:** допускается деликатный паттерн на основе элементов герба РТ / орнамента — только как фон секций с низкой непрозрачностью.

### 3.4. Доступность (обязательно для госсайта)

- Контраст текста ≥ WCAG AA (4.5:1).
- Полноценная «версия для слабовидящих»: увеличение шрифта, ч/б режим, отключение изображений.
- Навигация с клавиатуры, видимые focus-стейты, корректные ARIA-роли, alt-тексты.
- Адаптив: mobile-first, корректная работа на 360px+.
